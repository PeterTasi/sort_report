Insertion Sort

每次把一個元素插入左側已排序區間。用單步模式可看見元素如何右移，以及最後插入的位置。

平均時間複雜度 O(n^2) 空間複雜度 O(1) 近乎有序資料表現佳

演算法快速切換

Quick Sort Bubble Sort Selection Sort Insertion Sort Merge Sort
比較策略 與左側有序區間比較
交換行為 以右移方式騰出插入點
觀察重點 每輪擴大已排序區間
資料筆數 8
執行時間 0 ms
排序結果會顯示在這裡。

原理與 Big-O

Insertion Sort 會維持左側區間有序，逐步把新元素插入正確位置。資料越接近有序，執行效率越好。

最佳O(n)
平均O(n^2)
最差O(n^2)
空間O(1)
